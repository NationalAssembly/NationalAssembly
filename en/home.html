CENTRAL EUROPE INTERWAR FREEDOM TIMELINE

Discover the history of freedom in Central Europe after World War I through an interactive timeline and articles!



About the Project

We are a student-led international collaboration between the Kepler Gymnasium in Prague and the Bürgerwiese Gymnasium in Dresden, dedicated to exploring the history of freedom in Central Europe during the interwar period. Our project documents key moments of democratic progress, from the formation of Czechoslovakia to the Weimar Republic's constitutional developments. This website is entirely open-source, built with transparency and accessible to anyone interested in this pivotal era of European history.

About the Website

Please note: The site is currently in active development, with much of the content remaining as placeholder text and images or incomplete. Some features may produce errors or display unexpected results as content is being finalized.

The site uses JavaScript extensively to create an interactive timeline that synchronizes across multiple languages, handling language switching, dynamic content loading, and real-time iframe communication. JavaScript manages the responsive navigation, dropdown menus, and event listeners that power user interactions without requiring any server-side processing. The timeline itself responds to user input through postMessage events, creating a fluid experience even with embedded iframes. Additionally, the site features custom-themed cursors—historical figures from the period—that users can select and apply dynamically across the entire interface, stored locally in the browser without any external tracking. Built on GitHub Pages and designed to load efficiently, the site balances rich interactivity with accessibility, featuring responsive layouts and custom-built UI components that work seamlessly across browsers. We use no cookies or external analytics, respecting user privacy while maintaining full functionality.
